# Petals

**1년 전체를 한 화면에 보여주는 macOS 연간 캘린더 앱.**

Petals는 EventKit 캘린더 연동 위에 자유 배치 캔버스를 결합한 네이티브 SwiftUI 앱입니다. 스크롤 없이 한 윈도우에 1년 전체를 펼쳐 보고, 그 위에 이미지·텍스트·스티커·도형을 자유롭게 올려 나만의 비주얼 연간 달력을 만들 수 있습니다.

## 핵심 기능

### 연간 캘린더 모드 (기본)
- **연간 그리드 뷰** — 12행(월) × 31열(일)의 적응형 그리드. 윈도우 리사이즈 시 셀 크기 동적 재계산.
- **줌 레벨** — 12개월 / 3개월 / 1개월 뷰포트 전환.
- **EventKit 연동** — 사용자의 모든 캘린더(iCloud, Google, Exchange 등) 로드. 여러 날에 걸친 이벤트를 가로 컬러 바로 표시.
- **이벤트 CRUD** — 날짜 드래그로 생성, 팝오버로 조회/수정/삭제.
- **캘린더 필터** — 캘린더별 표시/숨김 토글.
- **오늘 라인** — 오늘 날짜를 전체 월에 걸쳐 세로선으로 표시.
- **캔버스 꾸미기** — 캘린더 위 투명 레이어에 이미지·텍스트·스티커·도형 배치. 이동·크기 조절·회전·Z 순서·불투명도 제어. 비율 좌표(0.0~1.0)로 저장해 리사이즈에도 위치 유지.
- **테마 시스템** — 배경·그리드 선·강조색을 제어하는 프리셋 테마.

### 화이트보드 모드
- 캘린더 그리드 없는 **무한 캔버스** 공간. 순수 창작·발상용.
- 줌/팬 제스처 지원, 여러 개의 독립 보드 관리.
- 캘린더 모드와는 완전히 분리된 데이터 구조 사용.

### 공통
- **키보드 단축키** — 새 이벤트, 오늘로 이동, 연도 이동, 복사/붙여넣기, 실행 취소/다시 실행, 이벤트 폰트 크기 조절 등.
- **다국어** — 한국어 / 영어.
- **CloudKit 동기화** — SwiftData 기반 자동 동기화. 실패 시 로컬 전용으로 폴백.

## 기술 스택

| 기술 | 용도 |
|------|------|
| Swift 5.0 | 주 언어 |
| SwiftUI | UI 프레임워크 |
| SwiftData | 캔버스·비전보드 데이터 영속 저장 |
| CloudKit | 기기 간 동기화 (SwiftData 연동) |
| EventKit | 캘린더 이벤트 접근 및 조작 |

- **플랫폼**: macOS 15.2 (Sequoia) 이상
- **배포**: Mac App Store

## 데이터 아키텍처

설계 원칙: EventKit 데이터는 복제하지 않고 EventKit API로 읽고 쓴다. 앱이 소유하는 데이터는 꾸미기 정보뿐이다.

SwiftData 모델 4종:

| 모델 | 역할 |
|------|------|
| `YearDocument` | 한 해의 캔버스 상태를 담는 루트 객체. `year`, `theme`, `canvasItems[]` 보유. |
| `CanvasItem` | 캘린더 위 장식 요소. 비율 좌표 + `zoomLevel`·`pageIndex`로 뷰포트 추적. |
| `VisionBoard` | 독립 무한 캔버스 보드. 뷰포트 상태(scale, offset)와 `items[]` 보유. |
| `VisionBoardItem` | 보드 위 요소. 절대 좌표 사용. |

캔버스/비전보드 아이템 타입은 공통으로 image · text · sticker · shape를 지원합니다. 이미지 원본은 파일로 분리 저장하고 SwiftData에는 썸네일만 둬 CloudKit 레코드 크기 제한에 대응합니다.

## 프로젝트 구조

```
Petals/Petals/Petals/
├── PetalsApp.swift          앱 진입점, SwiftData·CloudKit 초기화
├── ContentView.swift        캘린더/화이트보드 모드 전환 루트 뷰
├── Models/                  SwiftData 모델, 클립보드·스냅샷
├── Views/                   캘린더 그리드·이벤트 바·캔버스·비전보드·인스펙터
├── EventKit/                EventManager(권한·로드), EventLayoutEngine(배치)
├── Helpers/                 캘린더 레이아웃, 이미지 관리, 툴바 구성
├── Settings/                AppSettings, SettingsView
├── Theme/                   테마 정의
└── Localizable.xcstrings    한국어·영어 번역
```

## 빌드 및 실행

1. `Petals/Petals.xcodeproj`를 Xcode에서 엽니다.
2. macOS 15.2 이상을 타겟으로 빌드·실행합니다.
3. 첫 실행 시 EventKit(캘린더) 접근 권한을 요청합니다. 권한을 거부해도 빈 그리드 위 캔버스 기능은 사용할 수 있습니다.
